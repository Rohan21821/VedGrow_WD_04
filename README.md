# Weather Dashboard

A modern and responsive Weather Dashboard built using HTML, CSS, JavaScript, and OpenWeatherMap API.

## Features

* Search weather by city name
* Autocomplete city suggestions
* Current weather information
* 5-Day weather forecast
* Weather icons
* Celsius/Fahrenheit conversion
* Humidity, Wind Speed, Pressure, Visibility
* Sunrise & Sunset information
* Feels Like temperature
* Air Quality Index (AQI)
* Current Date & Time
* Dynamic weather backgrounds
* Current Location Weather (GPS)
* Dark / Light Mode
* Favourite Cities (Local Storage)
* Recent Searches
* Loading Indicator
* Error Handling
* Responsive Design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API
